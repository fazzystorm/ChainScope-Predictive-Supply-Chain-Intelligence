
# ChainScope: Predictive Supply Chain Intelligence

A high-performance visualization tool to analyze global dependency, market risks, and logistics infrastructure.


## The Problem

Global companies have complex dependencies. If one route closes, the entire market collapses. Existing tools are either too expensive or don't show the 'Full Picture' of a company's supply health.


## Features

Dependency Visualization: Clearly see the links between HQs, plants, and raw material sources.

Global Market Analysis: A bird’s-eye view of a company's trade routes (Sea, Rail, and Land).

Live Disruptions (Source: Global News): Integrated markers for events (like the Red Sea crisis) based on reports from BBC News and other global sources to show real-time impact.

High-Performance Mapping: Powered by MapTiler API for crisp, fast-loading vector tiles and custom map layers.




## Technical Stack

Frontend: HTML, CSS, JavaScript .

Mapping Engine: MapLibre GL JS with MapTiler Vector Tiles.

Data Sources: Global trade reports, AIS data structures, and news alerts (BBC/Global Media).

Design Philosophy: Minimalist Dark Mode to highlight critical data points.


## Future Goals

Transitioning from static dependency maps to dynamic AI-powered risk prediction.

Integrating real-time AIS shipping data via APIs.


## Why this matters?

I am a 17-year-old self-taught developer. Most tools for market analysis are cluttered and expensive. I built ChainScope to prove that we can visualize complex global dependencies simply and effectively. It’s not just a map; it’s a tool to understand the world's trade network.
